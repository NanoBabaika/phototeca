Проект выполнен в учебных целях.
  - Реализована возможность регистрации и логина пользователей.
  - Загрузка фото на сервер.
  - Загрузка аватара пользователя.
  - Пользователи могут просматривать фото друг друга.
  - Поиск пользователя и имени или городу.
  - Реализованы лайки и комментарии. 
  - Статистика профиля.
  - Восстановление пароля с письмом на почту.
Стек проекта: HTML, CSS, JS, PHP, MySQL
